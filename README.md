# ABSTRACTION-Automation-System

## Description
This project demonstrates the concept of **Abstraction** in Object-Oriented Programming (OOP) using Python.

An abstract class named `BuildingSystem` is created using Python's `abc` module. It defines three abstract methods:

1.start()
2.stop()
3.status()

These methods are implemented by different building systems.

## Classes
1.BuildingSystem (Abstract Class)
2.AirConditioningSystem
3.LightingSystem
4.SecuritySystem
5.FireAlarmSystem

## Features
- Uses Python's 'abc' module.
- Demonstrates abstraction through abstract classes and methods.
- Demonstrates polymorphism by storing objects in a list and calling the same methods on different objects.
- Easily extendable by adding new building system classes without modifying the existing processing loop.
